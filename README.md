Lab1 打地鼠小游戏项目简介
本项目为第二阶段课内实验，使用鸿蒙ArkUI框架开发一款休闲打地鼠互动小游戏。
1. 游戏规则：地鼠会随机出现在5个洞口内，在倒计时结束前点击出现的地鼠即可获得分数。
2. 功能实现：实时分数统计、倒计时计时器、随机位置生成、点击判定逻辑。
3. 界面布局：绿色草地背景、洞口方块、顶部信息栏与底部状态提示，完整还原小游戏交互体验。

Lab2 五子棋对弈项目简介
本项目为第三项课内实验，基于鸿蒙ArkUI开发双人五子棋对战程序。
1. 核心功能：黑白棋子交替落子、棋盘网格绘制、连成五子自动判定胜负并弹出胜利提示。
2. 附加功能：一键开启新对局，支持自由流转模式，随时重置棋局。
3. 逻辑设计：记录每一步落子坐标，横向、纵向、两条斜线四个方向自动检索五子连线，完成胜负判断。

Lab3 日程管理App项目简介
本项目是第四项实验，开发一款移动端日程待办管理工具，采用底部双标签页结构。
1. 页面模块：分为日程列表页与日历视图页，支持月/周/日三种浏览模式。
2. 核心功能：点击加号新增待办事项，填写任务内容、起止时间、重复周期与优先级。
3. 数据管理：统一管理所有日程任务，可筛选查看任务，空白状态下展示友好空页面提示。
4. 技术实现：使用ArkUI完成多页面切换、表单录入、日历控件渲染与本地数据存储。

Lab4 多端通讯社交软件项目简介
本项目为第五个课内实验，基于鸿蒙一次开发多端部署技术，开发仿即时通讯应用，支持跨设备自由流转。
1. 页面结构：底部导航包含消息、通讯录、朋友圈、我的四个标签页，当前展示会话列表。
2. 界面内容：展示联系人头像、昵称、最新消息与消息时间，支持顶部搜索与新建会话。
3. 关键技术：一套代码同时适配手机、平板设备；应用状态可跨设备无缝迁移，实现任务自由流转。
4. 数据展示：会话列表分页渲染，区分当日消息与历史消息，还原主流社交软件UI效果。
