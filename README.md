## 项目2：WhackAMole 打地鼠小游戏（Lab1）
- 项目简介：
  本项目是一个基于鸿蒙 ArkTS 开发的打地鼠小游戏，主要实现了以下功能：
  1.  游戏计时与倒计时功能，时间结束自动结束游戏。
  2.  点击地鼠得分系统，点击有效地鼠获得分数。
  3.  游戏状态控制，区分“未开始”“游戏中”“游戏结束”三种状态。
  4.  响应式 UI 布局，适配不同设备屏幕。

- 核心实现说明：
  - 使用 `gameState` 变量控制游戏流程。
  - 通过定时器实现倒计时和地鼠随机出现的逻辑。
  - 使用线性渐变模拟草地背景，提升视觉效果。

- 运行说明：
  使用 DevEco Studio 打开项目，在模拟器或真机上运行，点击“开始游戏”按钮即可开始体验。
